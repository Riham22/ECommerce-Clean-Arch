# E-Commerce Clean Architecture

Full e-commerce system built with:

- .NET 10 Web API
- Clean Architecture
- PostgreSQL / SQL Server
- Next.js (React) frontend (later Angular)
- Redis caching
- RabbitMQ
- Custom JWT auth
- Stripe payments
- Soft-delete everywhere
- Arabic / English support
- Light / Dark theme

## Structure

- **Domain** → Entities, value objects, domain events
- **Application** → Use cases, DTOs, interfaces
- **Infrastructure** → EF Core, Redis, RabbitMQ, Stripe
- **API** → Controllers, middleware

Work in progress — learning SOLID, OOD, system design step by step.
